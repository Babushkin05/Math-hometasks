$\begin{cases}
a_x + b_x+ c_x =0 \\\\
a_y + b_y + c_y = 0  \\\\
a_z + b_z + c_z = 0 
\end{cases}$

так как мы знаем длины :

$\begin{cases}
a_x^2 + a_y^2 + a_z^2 = 9 \\\\
b_x^2 + b_y^2 + b_z^2 = 1 \\\\
c_x^2 + c_y^2 + c_z^2 = 16
\end{cases}$

распишем $(a,b)+(b,c)$

$(a,b)+(b,c) = (a_xb_x + a_yb_y + a_zb_z) + (b_xc_x+b_yc_y + b_zc_z ) = 
b_x(a_x + c_x) + b_y (a_y + c_y) + b_z(a_z + c_z)$

$= -b_x^2 - b_y^2 - b_z^2 = -1$

аналогично для 

$(a,b)+(c,a) = -9$

$(b,c)+ (c,a) - -16$

тогда 

$(a,c)+(b,c)+(c,a) = \frac{-1-9-16}{2}=-13$
